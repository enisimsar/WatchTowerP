Watchtower API v1.2

Watchtower is a simple API allowing consumers to view the given theme’s influencers and feeds.

Change Log:

  • 22.08.2017

    • get_hashtag end point is added.
      1. themeid (int),
      2. themename,
      3. date(yesterday, week, month)
    • get_news end point is updated.
      1. topics (int, seperated with comma)

    05.08.2017

    • get_news end point parameters are increased.
      1. since(DAY-MONTH-YEAR),
      2. until(DAY-MONTH-YEAR),
      3. cities(seperated with comma),
      4. countries(seperated with comma),
      5. language(seperated with comma, language code),
      6. mention_location(seperated with comma),
      7. mention_language(seperated with comma, language code)
