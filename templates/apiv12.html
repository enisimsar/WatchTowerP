Watchtower API v1.2

Watchtower is a simple API allowing consumers to view the given topic’s audiences, events, conversations and news.

Change Log:

  • 23.08.2017

    • get_events end point is added.
      1. topic_id (int),
      2. date(date, interested),
      3. cursor (int)
    • get_conversations end point is added.
      1. topic_id (int),
      2. date(day, week, month),
      3. cursor (int)

    22.08.2017

    • get_hashtag end point is added.
      1. topic_id (int),
      2. date(yesterday, week, month)
    • search_news end point is updated.
      1. topic_ids (int, seperated with comma)

    05.08.2017

    • search_news end point parameters are increased.
      1. since(DAY-MONTH-YEAR),
      2. until(DAY-MONTH-YEAR),
      3. language(seperated with comma, language code),
      4. mention_location(seperated with comma),
      5. mention_language(seperated with comma, language code)
